# Spam-detection-project
This project classifies emails as spam or ham using machine learning


## 🧠 Models Used
- Naive Bayes
- Logistic Regression

## ⚙️ Features
- Text preprocessing
- TF-IDF vectorization
- Model comparison

## 📊 Results
Logistic Regression outperformed Naive Bayes with higher accuracy.

## 🚀 How to Run
Open the notebook in Google Colab and run all cells.
